Projeto: Gerenciamento de Hotéis

Detalhes do Projeto

O projeto "Gerenciamento de Hotéis" foi desenvolvido com o objetivo de criar uma plataforma completa de reservas de hotéis. A aplicação permite aos usuários visualizar quartos disponíveis, fazer reservas e gerenciar suas informações de maneira eficiente.

Tecnologias Utilizadas

  • HTML, CSS e JavaScript para o frontend
  • Node.js e Express para o backend
  • Banco de Dados MySQL para armazenamento de informações

Principais Funcionalidades

  • Cadastro e autenticação de clientes
  • Visualização de quartos disponíveis
  • Formulário de reservas online
  • Histórico de reservas
